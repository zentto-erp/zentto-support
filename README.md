# Zentto — Soporte técnico

Bienvenido al centro de soporte de **Zentto ERP**. Aquí puedes:

- 🐛 **Reportar un error** — Algo no funciona como debería
- ✨ **Solicitar una funcionalidad** — Sugiere mejoras
- ❓ **Hacer una pregunta** — Dudas sobre el sistema

## ¿Cómo abrir un ticket?

1. Haz clic en **[New Issue](../../issues/new/choose)**
2. Selecciona el tipo de solicitud
3. Completa el formulario
4. Nuestro equipo te responderá lo antes posible

## Tiempos de respuesta

| Plan | Primera respuesta | Resolución |
|------|------------------|------------|
| Enterprise | < 2 horas | < 24 horas |
| Profesional | < 8 horas | < 48 horas |
| Básico | < 24 horas | < 5 días |
